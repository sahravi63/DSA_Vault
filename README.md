# DSA Vault

A single-page DSA interview prep vault with C++ solutions, explanations, examples, search, category navigation, and difficulty filters.

Live site: https://sahravi63.github.io/DSA_Vault

## Features

- Search problems by title or category.
- Filter by difficulty: All, Easy, Medium, and Hard.
- Browse problems grouped by topic.
- Expand each problem to view code, logic, and sample input/output.
- Copy C++ solutions directly from the code panel.
- Responsive layout for desktop, tablet, and phone screens.

## Usage

Open `index.html` directly in a browser, or visit the live GitHub Pages link above.

No build step or dependencies are required.

## Mobile Support

The page includes phone-friendly styling with stacked controls, horizontal scrolling for compact chip groups, larger touch targets, wrapped problem titles, and scrollable code blocks.
